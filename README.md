


# TraktoTv

## Tecnologias utilizadas 
- Angular@13.2.4

## Rodar o projeto
```
git clone ![https://github.com/Roseffc/traktotv.git](https://github.com/Roseffc/traktotv.git)
cd traktotv
npm install 
ng serve

Abrir navegador: http://localhost:4200
```
